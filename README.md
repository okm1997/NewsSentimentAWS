# News Sentiment Analysis Using AWS

In this project we have used multiple services to create News providing API based on Sentiments
## Services Used
- AWS Lambda
- IAM
- DynamoDB
- Amazon Comprehend
- Secrets Manager
- EventBridge
- Amazon Cloudwatch
- API Gateway
- Postman
